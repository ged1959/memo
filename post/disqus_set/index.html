DISQUSをセットした

Mar 31, 2016   #disqus  #hugo 

コメントを残す方法。DISQUSを使う。ので、やった方法。

hugo側

angels-ladderなら簡単。config.tomlを以下のように書く。

baseurl = "http://ged1959.github.io/memo/"
languageCode = "ja-jp"
title = "My Memo"
canonifyurls = true
disqusShortname = "mymemo1959"

テーマの作者さんのgithubサイト「tanksuzuki/angels-ladder: Angel’s Ladder is a simple blog theme for Hugo.」にある通り。

DISQUSの設定
